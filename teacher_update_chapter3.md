# 第三章阶段性进展说明

## 一、目前已完成的内容

1. 已完成原始新闻数据接入与样本构建。原始语料来自 OpenNewsArchive 中文新闻数据，当前第三章 baseline 选取 `2023 年 11 月` 作为连续时间窗口，研究对象为 `宁德时代` 和 `贵州茅台` 两只股票。
2. 已完成新闻筛选与实验样本构造。当前样本采用“按日每股保留前 3 条最相关新闻”的方式构建，共 `180` 条新闻，其中宁德时代 `90` 条、贵州茅台 `90` 条。
3. 已完成文章级情绪分析与结构化。现在已经可以从单条新闻中抽取股票对象、事件类型、情绪得分、情绪强度、置信度、相关度、事件重要度和风险标记等字段。
4. 已完成股票级日频聚合与因子生成。当前已经能够将多条新闻按“股票-日期”聚合，生成 `net_sentiment_factor`、`negative_shock_factor`、`sentiment_momentum_factor`、`attention_factor` 和 `composite_score` 等股票级日频舆情因子。
5. 已完成真实 API baseline 实验。当前在 `180` 条新闻样本上完成了真实大模型实验，最终得到 `139` 条文章级结构化结果和 `58` 条股票日频画像结果。

## 二、目前做到哪一步

第三章目前已经完成到：

`新闻/舆情文本 -> 情绪分析 -> 数值化 -> 因子生成`

也就是说，因子生成这一步已经完成。

当前还没有完全展开的是：

`因子检验`

例如因子与未来收益率、股价走势、分组比较等有效性验证，准备作为下一步继续补充。

## 三、目前第三章的核心结论

1. 当前 baseline 已经验证了“从新闻文本到股票级日频舆情因子”的完整链路是可行的。
2. 相比“每天只保留 1 条新闻”的简化方案，当前采用“按日每股 top3”的方案，更能体现同日多条新闻聚合形成因子的过程。
3. 在当前样本窗口下，宁德时代和贵州茅台已经呈现出较明显的舆情差异，说明这套 baseline 具有一定的区分度和解释性。
